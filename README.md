## 🚀 Proje Hakkında
Moradam, React ve Tailwind CSS kullanılarak geliştirilen, içerisinde 42 farklı mini oyunu barındırmayı hedefleyen modern bir oyun portalıdır. En büyük özelliği, tüm geliştirme sürecinin bir PC üzerinden değil, tamamen bir **Samsung Galaxy Tab A9+** üzerinden yürütülmesidir.
### 🛠️ Geliştirme Ortamım (Mobile Dev)
* **Cihaz:** Samsung Galaxy Tab A9+ 📱
* **Araçlar:** Termux, Acode, MT Manager
* **Framework:** Next.js / React
* **Styling:** Tailwind CSS (Neon & Cyberpunk Style)
